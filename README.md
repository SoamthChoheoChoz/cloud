# **欢迎来到周韬宇的云盘!**
###### 此云盘由周韬宇和刘禛共同创作而成.

## 提交步骤
   1. 上传文件到File文件夹;
   2. 在"提交信息"处填写用途;
   3. 管理员检测.
### 可能遇到以下情况: 
   #### :worried: 不通过:
      😕不明用途:管理员删除文件,然后自行重新上传并重新填写用途.
   
      ❗️高危文件:取消查看此云盘的权限.

   #### :smile: 通过

      🎉  祝贺!你成功上传了一个文件! 🎉
