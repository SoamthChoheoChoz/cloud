### **欢迎来到周韬宇的云盘!**
你可以储存需要保存的文件.

禁止储存格式,用途不明的文件.

本站每一文件都经过管理员检测.

如发现高风险文件, **管理员会取消查看此云盘的权限** .

提交步骤
    上传文件;
    在"提交信息"处填写用途;
    管理员检测.
可能遇到以下情况: :worried: 不通过:

:confused: 不明用途:管理员删除文件,然后自行重新上传并重新填写用途.

:exclamation: 高危文件:取消查看此云盘的权限.

:smile: 通过
